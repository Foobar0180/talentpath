Edit Mail Template

There are a number of 'placeholders’ you can add to your mail templates. These are shown in the editor as labels inside square brackets e.g. [candidate_first_name]. They are replaced with the associated dynamic value (e.g. the candidate’s first name, your name or the position title) when the mail is sent.
